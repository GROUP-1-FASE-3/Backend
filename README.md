# Backend

API AIRBNB APPLICATION

About the project :
An application which enables users to both book a villas and provide villas for others to rent.

Features :
- Authentication
- User: 
    - Create User (Register New User)
    - Get User by ID
    - Delete User
    - Update User
- Villa :
    - Create Villa
    - Get all Villa (Homepage)
    - Get Villa by UserID
    - Update Villa
    - Delete Villa
   
- Reservation :
    - Check availability
    - Post Reservation
    - Get Reservation by user ID
   
- Ratings :
    - Get Ratings
    - Post Ratings
