# Backend

API AIRBNB APPLICATION

About the project :
An application which enables users to both book a room and provide rooms for others to rent.

Features :
- Authentication
- User: 
    - Create User
    - Get User by ID
    - Delete User
    - Update User
- Villa :
    - Create Villa
    - Get all Villa
    - Get Villa by id
    - Update Villa
    - Delete Villa
   
- Reservation :
    - Check availability
    - Post Reservation
    - Get Reservation by user ID
   
- Ratings :
    - Get Ratings
    - Post Ratings
